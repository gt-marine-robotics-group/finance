# Bill Request Automation

1. Download the images for the bill you are submitting on sharepoint and save them in the "/downloads" folder in the same directory as the .xlsx
2. Download the most recent bill .csv in UTF-8 encoding

3. Run automation.py.
    - If it gets stuck when saving items, click save and it should continue.


## Notes
- Images must have the same exact name as what is in the "Item Name" column.
